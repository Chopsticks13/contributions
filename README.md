# contributions

Auto-updating open source contributions page hosted at [chopsticks13.github.io/contributions](https://chopsticks13.github.io/contributions).

## How it works

A GitHub Action runs daily, fetches all public PRs by [@Chopsticks13](https://github.com/Chopsticks13) via the GitHub API, generates a static HTML page, and deploys it to GitHub Pages.

No frameworks, no build tools, no dependencies. Just Python stdlib + HTML + CSS.

## Manual update

Trigger the workflow manually from the [Actions tab](https://github.com/Chopsticks13/contributions/actions/workflows/update.yml).
